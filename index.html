One-time setup

This app keeps your stock in a free JSONBin.io bin so laptop and mobile stay in sync. No database, no localStorage.

  1. Create a free account at jsonbin.io.
  2. Create a Bin with content {"products":[],"movements":[]} and copy its Bin ID.
  3. On the API Keys page, create an Access Key scoped to that bin (Read + Update). Copy it.
  4. Open config.js and paste the Bin ID and Access Key.
  5. Reload this page. See README.txt for details.
