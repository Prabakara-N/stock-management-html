Needs to run on a server

This app talks to a small serverless function at /api/stock that keeps your JSONBin key secret. Opening the file directly (file://) has no server, so it can't sync.

To run it:

  1. Locally: npm i -g vercel then vercel dev in this folder, and open the printed http://localhost URL.
  2. Or deploy: push to GitHub, import the repo on vercel.com.
  3. Set env vars JSONBIN_MASTER_KEY and JSONBIN_BIN_ID (see README.txt).
